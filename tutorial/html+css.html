3标签的属性与值

首先键入html回车，就可以得到如下结构。


<!DOCTYPE html>
<html lang="en" dir="ltr">
    <head>
        <meta charset="utf-8">
        <title></title>
    </head>
    <body>
    </body>
</html>

这是一个完整的HTML文档的简单结构。


它像俄罗斯套娃一样，是一层一层嵌套的。
